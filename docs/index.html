TimeDSS

A VBA-powered Excel Decision Support System for planning, tracking, and optimizing how users allocate time across work, study, and personal tasks.

Download Workbook View on GitHub

Overview

TimeDSS transforms raw task and time-tracking inputs into actionable insights, productivity metrics, and weekly “lock-in” recommendations. Built entirely in Excel using VBA, it demonstrates how decision support systems can be implemented without external software dependencies.

Key Features

  • Task-based weekly planning with due dates and categories
  • Daily progress tracking and productivity scoring
  • Automated lock-in hour recommendations
  • Warnings for unrealistic schedules
  • Interactive VBA-driven dashboards and charts
